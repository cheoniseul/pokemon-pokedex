# pokemon-pokedex
PokeAPI 기반 포켓몬 도감 웹 애플리케이션

## 프로젝트 소개

- 제작 기간: 2025년 12월 중순 (약 2~3주)

> ⚠️ 본 프로젝트는 **상업적 목적이 아닌 개인 학습 및 취업 포트폴리오용**으로 제작되었습니다.  
> Pokémon 및 관련 데이터의 저작권은 **Nintendo / Game Freak / The Pokémon Company**에 있으며,  
> 본 프로젝트는 **비공식 개인 프로젝트**입니다.

이 프로젝트는 **PokeAPI**를 기반으로 포켓몬 데이터를 불러와  
카드 리스트 형태로 보여주고, 클릭 시 상세 정보를 확인할 수 있는  
**포켓몬 도감(Pokedex)** 웹 서비스입니다.

API 연동, 필터링, 모달 UI, 무한 스크롤, 반응형 레이아웃 등  
프론트엔드 개발에 필요한 핵심 기능을 직접 구현하며  
JavaScript 기반의 구조화된 로직 설계와 UI/UX 개선에 중점을 두었습니다.

---

## 주요 기능

- 포켓몬 전체 카드 리스트 출력  
- 무한 스크롤을 통한 포켓몬 데이터 로딩  
- 포켓몬 이름 **한글 검색**  
- 타입 / 지역 / 조건별 필터링  
- 포켓몬 카드 클릭 시 상세 정보 **모달 UI**  
- 타입별 컬러 시스템 적용  
- 모바일 / 태블릿 / PC 반응형 대응  

---

## 기술 스택

- HTML5 / CSS3 / JavaScript (ES6+)
- Fetch API  
- GitHub Pages

---

## 사용 API

- **PokeAPI**  
  https://pokeapi.co/  
  포켓몬 기본 정보, 타입, 종족(Species) 데이터 사용  

---

## 프로젝트에서 신경 쓴 부분

- API 호출 최소화를 위한 데이터 캐싱 처리  
- 검색 / 필터 / 카드 리스트 로직 분리  
- UI 상태와 데이터 상태의 일관성 유지  
- 모달 UI와 카드 UI 간의 자연스러운 사용자 흐름 설계  
- 모바일 환경에서도 사용하기 쉬운 UX 구성  
- 유지보수를 고려한 JavaScript 모듈화
